# Data Professionals Survey
This project is a survey carried out on data professional to understand the industry with a few to providing information for aspiring data professionals.
The project showcases my skills in Microsoft SQL Server, Visualization with Tableau and PowerBI; data exploration and transformation with PowerQuery, DAX and Excel.
